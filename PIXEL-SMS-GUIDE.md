## 🚀 **COMMENT UTILISER AVEC TON PIXEL**

### **OPTION 1: Interface HTML (RECOMMANDÉE)**

**Sur ton Google Pixel:**

```
1. Transfère le fichier HTML sur ton Pixel
   - Par email
   - Par USB
   - Par Google Drive

2. Ouvre le fichier HTML dans Chrome

3. Pour chaque lead:
   - Clique sur "📤 Send SMS"
   - L'app SMS s'ouvre avec:
     • Numéro pré-rempli
     • Message pré-rempli
   - Clique "Envoyer"
   - Retour à la page
   - Clique sur le suivant

4. Marque comme envoyé dans le CSV
```

**Avantages:**
- ✅ 1 clic = 1 SMS
- ✅ Message automatique
- ✅ Numéro automatique
- ✅ Interface propre

---

### **OPTION 2: CSV + Copier-Coller**

```
1. Ouvre le CSV sur ton Pixel
2. Copie le numéro de téléphone
3. Ouvre l'app SMS
4. Colle le numéro
5. Copie le template de message
6. Envoie
7. Passe au suivant
```

---

### **OPTION 3: Liste Texte**

```
1. Ouvre sms-pixel-*-phones.txt
2. Copie les numéros par batch de 50
3. Envoie SMS groupés
```
