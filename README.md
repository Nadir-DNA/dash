# ⬡ Dash

> Nothing Glyph dashboard — SMS campaigns, CRM, leads management

## Quick Start

```bash
cd ~/projects/dash
npm install
cp .env.example .env
npm run dev
```

## Stack

- **Framework** : Next.js 15 (App Router)
- **DB** : TrailBase (SQLite)
- **Design** : Nothing Phone Glyph dark theme
- **Components** : shadcn/ui + custom Glyph components
